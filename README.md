### 版本特性：

- 壁纸核心端
  - [x] 提供 API 驱动壁纸核心功能
  - [x] 开机启动核心端即可运行壁纸
  - [x] 弱 UI 减少资源占用，仅提供托盘
  - [ ] 增加exe壁纸支持
- UI 端
  - [x] 社区分享，此功能 UI 端提供
    - [x] 壁纸投稿
  - [x] Nuxt+TCB 实现，暂未开源

### 编译环境：

- VS2019
- Win10

### 分支管理
- master 是最新版代码（当前是2.x)，新版本商店名为 巨应壁纸，老版本尚未更新（巨应动态壁纸）
- 1.x 目前线上WPF开发的版本

### 注意事项
- 新版本巨应壁纸需要使用本地5001端口进行通信，请注意浏览器拦截(brave浏览器，及其他安全插件）
